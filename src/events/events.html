10th Feb 2023

GDSC Web Development & Design Workshop- Day 2

The GDSC workshop focused on TailwindCSS and Git/GitHub introduction on day two. Participants learned the basics of setting up TailwindCSS in a project and using its classes to style web pages. GDSC Web Development & Design Workshop- Day 1

The GDSC workshop's second day was dedicated to web development fundamentals, covering HTML and CSS. Attendees learned to structure web pages and style them with CSS, and were introduced to version control, making it an excellent starting point for novice web developers.

10th Feb 2023

Tech Talk: How to make coding effortless by Ankit Keshav

Ankit Keshav gave a tech talk on making coding effortless, covering a range of tips and tricks to streamline the coding process. He discussed the importance of tools such as code editors, keyboard shortcuts, and code snippets, and emphasized the value of good coding habits. Attendees left with practical advice on how to optimize their workflow and become more efficient coders.

10th Feb 2023

Android Study Jams: New to Programming

Our Android App Development event offered hands-on experience with Jetpack Compose, Kotlin, and Compose. Participants earned Google DSC certificates and kickstarted their journey in Android development with beginner-friendly workshops. They networked with experts and peers and enhanced their skills to boost their careers. We hope they found the event valuable and wish them all the best in their future endeavors.

10th Feb 2023

Workshop: Intro to Firebase and Full Stack Web Development

GDSC conducted an event titled "Intro to Firebase and Full Stack Web Development." The workshop provided attendees with an introduction to Firebase and taught them how to build full-stack web applications. Participants learned about the fundamentals of Firebase, including Realtime Database, Authentication, and Hosting. The event was a great success, and attendees left with practical knowledge and skills in full-stack web development.
